# 手動テスト項目一覧

## テスト環境の準備

### 前提条件
- ローカルサーバーが起動していること（Live Server等）
- ブラウザのDevToolsを開いておくこと（コンソールエラー確認用）

### サーバー起動方法
```bash
# VS Code Live Server の場合
# index.html を右クリック → "Open with Live Server"

# または Python の場合
python -m http.server 5500
```

## 注意事項

1. **LocalStorageのクリア方法**
   - DevTools → Application → Local Storage → 該当ドメイン → 右クリック → Clear

2. **コンソールエラーの確認**
   - 各テスト後、DevToolsのConsoleタブで赤いエラーがないことを確認

3. **キャッシュクリア**
   - 動作がおかしい場合は `Ctrl+Shift+R` でハードリロード
